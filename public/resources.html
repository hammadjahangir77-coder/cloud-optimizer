Illustrative numbers for a small application server — not a live quote. Real prices change by region, currency, and contract.

Pricing style What it means Example monthly cost Good for…
On-demand (pay as you go) Pay for each hour or second you use. Stop anytime. ~240 (example units) Tests, short projects, unpredictable traffic
Reserved / 1-year plan Promise to keep a baseline size for a year; provider lowers the rate. ~140 (example units) Steady websites or databases you know will run all year
3-year commitment (deepest discount) Longer promise — often the largest % off in vendor materials. ~95 (example units) Mature systems with predictable load (vendor docs cite up to ~72% vs on-demand)

5-minute cloud audit

  1. Set budget alerts — get an email before the bill surprises you.
  2. Delete unused backups (old snapshots) that you no longer need.
  3. Find idle systems running 24/7 with almost no use.
  4. Check "orphan" storage — disks not attached to any active computer.
  5. Schedule off-hour shutdowns for test/training environments.
